Títulos y parrfos

La etiqueta <p> </p> sirve para párrafos

La etiqueta <br/> sirve para insertar un salto de linea cuando me interese

La etiqueta <hr/> sirve para insertar una linea horizontal en el texto

Los títulos se pueden descartar haciendo uso de las etiquetas <h1>, <h2>, <h3> ... <h6>

Dando formato al texto

Usamos la etiqueta <strong> y </strong> para destacar una parte del texto. Normalmente el texto incluido entre esas etiquetas se representa en negrita

Usamos la etiqueta <em> y </em> para poner el texto en cursiva

Listas en HTML

Tenemos dos tipos: ordenadas y desordenadas. Para ambas, los elementos que contienen deben ir entre las etiquetas <li>

Listas ordenadas

Para las listas ordenadas usamos la etiqueta <ol>.

<ol reversed>

  1. primero
  2. segundo
  3. tercero

<ol type="A">

  1. primero
  2. segundo
  3. tercero

<ol type="a">

  1. primero
  2. segundo
  3. tercero

<ol type="I">

  1. primero
  2. segundo
  3. tercero

<ol type="i">

  1. primero
  2. segundo
  3. tercero

<ol type="1">

  1. primero
  2. segundo
  3. tercero

Listas desordenadas

Para las listas desordenadas usamos la etiqueta <ul>.

<ul type="circle">

  • primero
  • segundo
  • tercero

<ul type="disc">

  • primero
  • segundo
  • tercero

<ul type="square">

  • primero
  • segundo
  • tercero

<ul type="none">

  • primero
  • segundo
  • tercero

Hiperenlaces

Enlaces a rutas absolutas

Para crear un enlace usamos la etiqueta <a> </a> con el atributo href que determina el destino del enlace

Ejemplo: pulsa en este texto para ir a W3Schools

Enlaces a rutas relativas

Cuando queremos acceder a un recurso que estas en el mismo nivel o referente al nivel que estamos viendo, podemos usar enlaces relativos.
